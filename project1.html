Mobile App Redesign

Improving usability and navigation for users

Project Overview

Role: UI/UX Designer

Tools: Figma, FigJam

Duration: 3 Weeks

Problem Statement

Users were struggling to complete tasks due to confusing navigation, poor information hierarchy, and inconsistent UI elements.

User Research

  • Users wanted faster navigation
  • Important actions were hard to find
  • Too many steps to complete simple tasks

Wireframes & User Flow

Low-fidelity wireframes were created to simplify navigation and reduce cognitive load.

Final UI Design

The final design focuses on clarity, consistency, and accessibility.

Outcome & Learnings

The redesign improved task completion speed and overall user satisfaction. This project strengthened my understanding of user-centered design.
